   SCSS-07 요구사항 충족: 로드 순서가 한국어 주석으로 문서화된다.
   SCSS-01 요구사항 충족: Bootstrap 5 SCSS를 임포트하는 표준 진입점 파일이 존재한다.

   **중요한 구조적 제약:**
   - `@use` 문은 파일 최상단에 위치해야 한다 (Sass 규칙)
   - `@import`는 `@use` 뒤에 위치한다
   - Bootstrap(@import)은 1-settings(@use) 뒤에 와야 $font-size-base 오버라이드가 적용된다
   - 3-generic/_vendor.scss는 @import 기반이므로 @use '3-generic'이 아닌 @import '3-generic/vendor'로 불러온다

   ```scss
   // ====================================================
   // artpqUX 퍼블리싱 가이드 — 메인 진입점
   //
   // 로드 순서 (ITCSS 기반):
   //   1-settings  → 변수, 토큰 (CSS 출력 없음)
   //   2-tools     → 믹스인, 함수 (CSS 출력 없음)
   //   3-generic   → Bootstrap + reset (요소 선택자)
   //   4-elements  → HTML 태그 기본 스타일
   //   5-objects   → 레이아웃 패턴
   //   6-components → UI 컴포넌트
   //   7-utilities → 유틸리티 클래스
   //
   // Bootstrap 임포트 전략:
   //   Bootstrap 5.3.x는 내부적으로 @import 사용.
   //   3-generic/_vendor.scss에 @import로 격리하고,
   //   나머지 레이어는 @use/@forward 기반으로 유지한다.
   //   $font-size-base 등 오버라이드 변수는 1-settings에서 선언하여
   //   Bootstrap 로드 이전에 적용되도록 한다.
   // ====================================================

   // 1. 설정 (Bootstrap 오버라이드 변수 포함 — 반드시 Bootstrap보다 먼저)
   @use '1-settings' as settings;

   // 2. 도구 (믹스인, 함수)
   @use '2-tools' as tools;

   // 3. Bootstrap (@import 격리 — @use/@forward 경계 밖)
   // _vendor.scss 내부가 @import 기반이므로 @use 불가
   @import '3-generic/vendor';

   // 4. HTML 요소 기본 스타일
   @use '4-elements' as elements;

   // 5. 레이아웃 오브젝트
   @use '5-objects' as objects;

   // 6. UI 컴포넌트
   @use '6-components' as components;

   // 7. 유틸리티
   @use '7-utilities' as utilities;
   ```
