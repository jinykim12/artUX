```scss
// ====================================================
// artpqUX 팀 표준 브레이크포인트
//
// Bootstrap $grid-breakpoints (기본값 유지 — D-07):

//   태블릿   768~1023px Bootstrap md 시작점과 동일
//   소형 PC 1024~1279px Bootstrap 기준에 없는 팀 전용 구간
//   PC      1280px~
//
// Bootstrap 그리드 클래스(col-md-6 등)는 Bootstrap 기본 breakpoint 동작.
// respond-to()는 커스텀 컴포넌트 전용.
// ====================================================
```

3. **$breakpoints Sass Map** (per D-04):
```scss
$breakpoints: (
  mobile-max:  767px,
  tablet:      768px,
  tablet-max:  1023px,
  pc-sm:       1024px,
  pc-sm-max:   1279px,
  pc:          1280px,
) !default;
```

4. **respond-to() 믹스인** (per D-06) — `_mixin.scss`와 동일한 한국어 블록 주석 패턴 사용:
   - 주석 블록: 모바일 퍼스트 원칙 설명 (per D-10), 예외 허용 안내 (per D-11, D-12), 사용 예시 코드, 파라미터 설명
   - 구현: `$min-keys: (tablet, pc-sm, pc)` 리스트와 `$only-map` 맵으로 분기
   - min-width용 3키워드: `tablet` -> `@media (min-width: 768px)`, `pc-sm` -> `@media (min-width: 1024px)`, `pc` -> `@media (min-width: 1280px)`
   - range용 3키워드: `mobile-only` -> `@media (max-width: 767px)`, `tablet-only` -> `@media (min-width: 768px) and (max-width: 1023px)`, `pc-sm-only` -> `@media (min-width: 1024px) and (max-width: 1279px)`
   - 유효하지 않은 키워드 시 `@error` 발생 (per D-05)
