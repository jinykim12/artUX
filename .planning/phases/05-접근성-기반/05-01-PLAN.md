1. `scss/4-elements/_focus.scss` 신규 생성 (per D-01, D-02, D-03, D-04):

```scss
// ====================================================
// [포커스 스타일 표준 — A11Y-03]
// 이동: scss/2-tools/_mixin.scss → scss/4-elements/_focus.scss
//
// CSS 변수 호환 주의:
//   Sass rgba($var, $alpha) 불가 — CSS 변수는 컴파일 타임에 값이 없음
//   대신 outline: 2px solid var(--color-primary) 단독 사용
//
// 고대비 모드(forced-colors):
//   outline 기반 포커스 인디케이터는 forced-colors에서 자동 대응됨
//   별도 @media (forced-colors: active) 코드 불필요 (per D-03)
// ====================================================

/// 포커스 스타일 믹스인
/// @param {Length} $offset [-0.4rem] - outline-offset 값
/// 사용 예: @include focus();
///          @include focus($offset: 2px);
@mixin focus($offset: -0.4rem) {
  outline-offset: $offset;
  outline: 2px solid var(--color-primary);
}

// 전역 :focus-visible 적용
// :focus 대신 :focus-visible 사용 — 마우스 클릭 시 outline 미표시
// (키보드/보조기술 사용 시에만 outline 표시)
*:focus-visible {
  @include focus();
}

// :focus-visible 미지원 환경 폴백 (per KWCAG 준수)
// 구형 브라우저에서는 :focus로 폴백
*:focus:not(:focus-visible) {
  outline: none;
}
```

  top: 0;
  left: 0;
