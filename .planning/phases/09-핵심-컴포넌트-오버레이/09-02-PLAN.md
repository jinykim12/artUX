3. **핵심 ARIA 속성 표** — 속성/값/역할 3열 표:
- aria-label (슬라이더 목적 설명)
- aria-live="polite" (현재 슬라이드 변경 상태 공지)
- aria-atomic="true" (슬라이드 번호 전체 읽음)
- hidden (비활성 버튼 숨김 처리)

4. **자동 재생 정지/재생 버튼 JS 예시 (per D-08)**:
```javascript
// Swiper 자동 재생 + 정지/재생 버튼 + aria-live 상태 공지
const slider = document.querySelector('.slider');
const statusEl = slider.querySelector('.slider__status');
const pauseBtn = slider.querySelector('.slider__pause');
const playBtn = slider.querySelector('.slider__play');

const swiper = new Swiper('.swiper', {
  loop: true,
  autoplay: { delay: 5000, disableOnInteraction: false },
  a11y: {
    prevSlideMessage: '이전 슬라이드',
    nextSlideMessage: '다음 슬라이드',
    paginationBulletMessage: '{{index}}번 슬라이드로 이동',
  },
  keyboard: { enabled: true },
  on: {
    slideChange(swiper) {
      // aria-live 상태 갱신
      const current = swiper.realIndex + 1;
      const total = swiper.slides.length;
      if (statusEl) {
        statusEl.textContent = `${current} / ${total}`;
      }
    }
  }
});

// 정지 버튼
pauseBtn.addEventListener('click', () => {
  swiper.autoplay.stop();
  pauseBtn.hidden = true;
  playBtn.hidden = false;
  playBtn.focus();
});

// 재생 버튼
playBtn.addEventListener('click', () => {
  swiper.autoplay.start();
  playBtn.hidden = true;
  pauseBtn.hidden = false;
  pauseBtn.focus();
});
```

5. **Swiper.js 접근성 옵션 표** — 주요 옵션 정리:
- `a11y.prevSlideMessage` / `a11y.nextSlideMessage` (이전/다음 버튼 aria-label)
- `a11y.paginationBulletMessage` (페이지네이션 점 aria-label, `{{index}}` 사용)
- `keyboard.enabled: true` (키보드 탐색 허용)
- `loop: true` 사용 시 스크린리더 중복 슬라이드 주의 사항
